## Watch-outs specific to this codebase

- Supabase's generated types model one-to-one relations as arrays. Always cast via `as unknown as <Shape>[]` after selects that use nested joins.
- Next.js layouts don't receive `searchParams` in Next 16. The topbar reads `fc_active_campaign` from cookies as the fallback; page-level `?c=<uuid>` still wins for page content.
- `"use server"` files can ONLY export async functions. Type exports ‚Üí sibling `-types.ts` file.
- Agent-browser screenshots MUST use `--full` or you capture only the viewport; viewport-only crops on 8000px-tall pages have caused multiple false "parity ‚úì" claims in this repo.
